# 千企AI

#### 介绍
千企AI是集大模型能力接入、使用、管理于一体的一站式AI应用开发平台， 集成业界主流的对话模型、嵌入模型以及多模态模型，为用户提供智能知识库、客服、聊天助手、智能体、工作流等AI能力。
平台致力于降低AI能力使用门槛，让更多的中小企业可以低成本甚至0成本落地AI能力，满足公司各种业务场景的需要。
